What is String?

Let's us first know about the Fundamentals of characters

Characters are building blocks of programs.
Every program is a sequence of meaningfully grouped characters.

Character constant

An int value represented as a character in single quotes.
'z' represents the integer value of z (122 ASCII value).

Fundamentals of Strings

  • Array of characters treated as a single unit called string.
  • It can include letters, digits and special characters (*, /,$).
  • String literal (string constant) - written in double quotes.
      • I Love my Country.
  • Strings are arrays of characters
    1. String is a pointer to first character (like array).
    2. Value of string is the address of first character.
  • Each element of the string is stored in a contiguous memory locations.
  • It is terminated by a null character('\0') which is automatically inserted by the compiler to indicate the end of string.

To sort the strings in increasing order.
That is if list of names is given then sort the list in alphabetical order.
Use strcmp() and strcpy() functions.

Program to sort the strings using arrays.

#include<stdio.h>
#include<conio.h>
#include<string.h>
void main()
{
	char name[20][20];
	char temp[20];
	int i,j,n,l;
	printf("Enter the no. of string to be sorted:");
	scanf("%d",&n);
	printf("Enter %d strings:\n", n);
		for(i=0;i<=n;i++)
			gets(name[i]);
		for(i=0;i<=n;i++)
			{
			for(j=i+1;j<=n;j++)
				{
				l = strcmp(name[i], name[j]);
				if(l>0) // if first string is greater then swap
				{
					strcpy(temp, name[i]);
					strcpy(name[i],name[j]);
					strcpy(name[j],temp);
				}
			}
	}
 puts("Sorted list is:\n");
 for(i=0;i<=n;i++)
	puts(name[i]);
 getch();
}
