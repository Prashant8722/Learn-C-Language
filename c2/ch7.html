General form: type array_name[size] = { list of values }; n Examples: int marks[5] = {72, 83, 65, 80, 76}; char name[4] = {"A", "m", "i", "t"}; n The size may be omitted. In such cases the compiler automatically allocates enough space for all initialized elements int flag[ ] = {1, 1, 1, 0}; char name[ ] = {"A", "m", "i", "t"};
